# Zsh

Extended Bourne shell with many improvements.

## Technology

- **Tool**: Z shell (zsh)
- **Category**: Shell
- **Language**: C
- **Purpose**: Interactive shell with advanced features

## Installation Method

Installs via Homebrew.

## Notes

Zsh is the default shell on macOS since Catalina.
