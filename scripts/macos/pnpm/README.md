# pnpm

Fast, disk space efficient package manager for Node.js.

## Technology

- **Tool**: pnpm
- **Category**: Package Manager
- **Language**: JavaScript
- **Purpose**: Install npm packages with a content-addressable store

## Installation Method

Installs via Homebrew.
