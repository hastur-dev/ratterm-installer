# gpustat

Simple GPU status monitoring utility.

## Technology

- **Tool**: gpustat
- **Category**: System Monitoring
- **Language**: Python
- **Purpose**: Monitor NVIDIA GPU usage and statistics

## Installation Method

Installs via pip (requires Python).

## Notes

Designed for NVIDIA GPUs; limited functionality on macOS without NVIDIA hardware.
