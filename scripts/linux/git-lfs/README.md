# Git LFS

Git extension for versioning large files.

## Technology

- **Tool**: Git Large File Storage (LFS)
- **Category**: Git, Version Control
- **Language**: Go
- **Purpose**: Track large files in Git repositories efficiently

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to pacman (Arch Linux)
4. Falls back to Homebrew
