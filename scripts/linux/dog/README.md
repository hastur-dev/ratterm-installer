# dog

Command-line DNS client with colorful output, similar to dig.

## Technology

- **Tool**: dog
- **Category**: Networking, DNS
- **Language**: Rust
- **Purpose**: DNS lookups with human-friendly output

## Installation Method

1. Attempts installation via cargo
2. Falls back to pacman (Arch Linux)
3. Falls back to direct binary download from GitHub releases
