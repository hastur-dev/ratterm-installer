# VHS

CLI for creating terminal GIFs.

## Technology

- **Tool**: VHS
- **Category**: CLI Utilities
- **Language**: Go
- **Purpose**: Record terminal sessions as animated GIFs

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to GitHub releases binary download

## Notes

Supports x86_64 and arm64 architectures.
