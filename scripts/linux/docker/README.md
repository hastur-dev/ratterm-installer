# Docker

Container platform for building, shipping, and running applications.

## Technology

- **Tool**: Docker
- **Category**: Container, DevOps
- **Language**: Go
- **Purpose**: Application containerization and orchestration

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
