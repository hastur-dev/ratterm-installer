# k3d

Lightweight wrapper to run k3s (Kubernetes) in Docker.

## Technology

- **Tool**: k3d
- **Category**: Container/Kubernetes
- **Language**: Go
- **Purpose**: Create local Kubernetes clusters using Docker containers

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to official install script
