# bat

A cat clone with syntax highlighting and Git integration.

## Technology

- **Tool**: bat
- **Category**: CLI Utilities
- **Language**: Rust
- **Purpose**: Enhanced file viewing with syntax highlighting

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu - installs as `batcat`)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)

## Notes

On Debian/Ubuntu, the command is `batcat` due to a name conflict. Create an alias: `alias bat=batcat`
