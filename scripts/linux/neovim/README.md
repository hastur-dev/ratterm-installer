# Neovim

Hyperextensible Vim-based text editor.

## Technology

- **Tool**: Neovim
- **Category**: Text Editor
- **Language**: C, Lua
- **Purpose**: Modern text editor with Lua-based configuration

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
