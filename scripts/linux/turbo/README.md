# Turborepo

High-performance build system for JavaScript and TypeScript monorepos.

## Technology

- **Tool**: Turborepo
- **Category**: Build Tools
- **Language**: Rust
- **Purpose**: Incremental builds and caching for monorepos

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to npm global install
