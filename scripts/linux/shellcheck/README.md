# ShellCheck

Static analysis tool for shell scripts.

## Technology

- **Tool**: ShellCheck
- **Category**: Linting
- **Language**: Haskell
- **Purpose**: Find bugs and improve shell scripts

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to pacman (Arch Linux)
4. Falls back to Homebrew
