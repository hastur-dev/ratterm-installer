# xsv

Fast CSV command-line toolkit.

## Technology

- **Tool**: xsv
- **Category**: Data Processing
- **Language**: Rust
- **Purpose**: Index, slice, search, and join CSV files

## Installation Method

1. Attempts installation via pacman (Arch Linux)
2. Falls back to Cargo installation
