# grex

Command-line tool for generating regular expressions from user-provided input.

## Technology

- **Tool**: grex
- **Category**: CLI Utilities
- **Language**: Rust
- **Purpose**: Generate regex patterns from example strings

## Installation Method

1. Attempts installation via cargo
2. Falls back to pacman (Arch Linux)
3. Falls back to direct binary download from GitHub releases
