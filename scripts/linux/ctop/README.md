# ctop

Top-like interface for container metrics.

## Technology

- **Tool**: ctop
- **Category**: Container, Monitoring
- **Language**: Go
- **Purpose**: Real-time metrics for running containers (Docker, runC, etc.)

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to direct binary download from GitHub releases
