# gpustat

Simple command-line utility for querying and monitoring GPU status.

## Technology

- **Tool**: gpustat
- **Category**: Monitoring, GPU
- **Language**: Python
- **Purpose**: Monitor NVIDIA GPU utilization, memory, and temperature

## Installation Method

1. Installs via pip (python -m pip install gpustat)
