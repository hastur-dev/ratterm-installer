# tmux

Terminal multiplexer for managing multiple terminal sessions.

## Technology

- **Tool**: tmux
- **Category**: Terminal
- **Language**: C
- **Purpose**: Create and manage multiple terminal sessions

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
