# Ruby

Dynamic, open source programming language.

## Technology

- **Tool**: Ruby
- **Category**: Programming Language
- **Language**: C
- **Purpose**: General-purpose scripting and web development

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)

## Notes

Installs both `ruby` and development headers. Includes `gem` package manager.
