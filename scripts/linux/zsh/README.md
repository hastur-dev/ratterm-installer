# Zsh

Extended Bourne shell with many improvements.

## Technology

- **Tool**: Z shell (zsh)
- **Category**: Shell
- **Language**: C
- **Purpose**: Interactive shell with advanced features

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
