# JavaScript (Node.js)

JavaScript runtime environment based on Chrome's V8 engine.

## Technology

- **Tool**: Node.js
- **Category**: Runtime
- **Language**: C++/JavaScript
- **Purpose**: Execute JavaScript code outside the browser

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)

## Notes

Installs both `nodejs` and `npm` packages.
