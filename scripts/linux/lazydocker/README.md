# lazydocker

Terminal UI for managing Docker containers and images.

## Technology

- **Tool**: lazydocker
- **Category**: Docker, TUI
- **Language**: Go
- **Purpose**: Simple terminal dashboard for Docker

## Installation Method

1. Attempts installation via dnf with COPR (Fedora)
2. Falls back to pacman (Arch Linux)
3. Falls back to official install script
