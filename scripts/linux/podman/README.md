# Podman

Daemonless container engine for developing, managing, and running OCI Containers.

## Technology

- **Tool**: Podman
- **Category**: Container
- **Language**: Go
- **Purpose**: Docker-compatible container runtime without daemon

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to pacman (Arch Linux)
4. Falls back to Homebrew
