# Make

GNU Make build automation tool.

## Technology

- **Tool**: GNU Make
- **Category**: Build Tools
- **Language**: C
- **Purpose**: Build automation using Makefiles

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu - with build-essential)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
