# ncdu

NCurses Disk Usage analyzer.

## Technology

- **Tool**: ncdu
- **Category**: Disk Utilities
- **Language**: C
- **Purpose**: Interactive disk usage analyzer with ncurses interface

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum with EPEL (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
