# Helix

Post-modern modal text editor with built-in LSP support.

## Technology

- **Tool**: Helix (hx)
- **Category**: Text Editor
- **Language**: Rust
- **Purpose**: Terminal-based editor with tree-sitter and LSP integration

## Installation Method

1. Attempts installation via apt-get with PPA
2. Falls back to dnf with COPR
3. Falls back to pacman (Arch Linux)
4. Falls back to AppImage download
