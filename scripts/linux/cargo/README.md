# Cargo (Rust Toolchain)

Rust's package manager and build system, installed via rustup.

## Technology

- **Tool**: Cargo, rustc, rustup
- **Category**: Build Tools, Package Manager
- **Language**: Rust
- **Purpose**: Compile Rust code and manage dependencies

## Installation Method

1. Uses official rustup installer from rustup.rs
2. Installs complete Rust toolchain (cargo, rustc, rustup)
