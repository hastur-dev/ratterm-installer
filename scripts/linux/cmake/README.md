# CMake

Cross-platform build system generator for C/C++ and other languages.

## Technology

- **Tool**: CMake
- **Category**: Build Tools
- **Language**: C++
- **Purpose**: Generate native build files for various platforms and compilers

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
