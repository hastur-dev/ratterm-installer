# kind

Kubernetes IN Docker - run local Kubernetes clusters using Docker containers.

## Technology

- **Tool**: kind
- **Category**: Kubernetes, Container
- **Language**: Go
- **Purpose**: Run local Kubernetes clusters for development and testing

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to `go install`
3. Falls back to direct binary download
