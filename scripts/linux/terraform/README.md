# Terraform

Infrastructure as Code tool for building, changing, and versioning infrastructure.

## Technology

- **Tool**: Terraform (HashiCorp)
- **Category**: Infrastructure, DevOps
- **Language**: Go
- **Purpose**: Manage cloud infrastructure declaratively

## Installation Method

1. Attempts installation via apt-get with HashiCorp repository
2. Falls back to dnf/yum with HashiCorp repository
3. Falls back to pacman (Arch Linux)
