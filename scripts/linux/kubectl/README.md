# kubectl

Kubernetes command-line tool for managing clusters.

## Technology

- **Tool**: kubectl
- **Category**: Kubernetes
- **Language**: Go
- **Purpose**: Control Kubernetes clusters via command line

## Installation Method

1. Attempts installation via apt-get with Kubernetes repository
2. Falls back to pacman (Arch Linux)
3. Falls back to direct binary download
