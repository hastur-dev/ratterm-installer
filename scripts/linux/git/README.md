# Git

Distributed version control system for tracking changes in source code.

## Technology

- **Tool**: Git
- **Category**: Version Control
- **Language**: C
- **Purpose**: Source code version control and collaboration

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
