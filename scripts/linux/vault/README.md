# Vault

## Technology

- **Tool**: HashiCorp Vault
- **Category**: Security
- **Language**: Go
- **Purpose**: Manage secrets, encryption, and access control

1. Attempts installation via HashiCorp APT repository (Debian/Ubuntu)
2. Falls back to Homebrew
