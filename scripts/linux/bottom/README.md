# bottom

Cross-platform graphical process/system monitor with a customizable interface.

## Technology

- **Tool**: bottom (btm)
- **Category**: System Monitoring
- **Language**: Rust
- **Purpose**: Interactive system monitor similar to htop/top

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to pacman (Arch Linux)
4. Falls back to cargo install
