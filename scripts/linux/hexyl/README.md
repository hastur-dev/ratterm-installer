# hexyl

Command-line hex viewer with colored output.

## Technology

- **Tool**: hexyl
- **Category**: CLI Utilities
- **Language**: Rust
- **Purpose**: View file contents in hexadecimal format

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to pacman (Arch Linux)
3. Falls back to cargo install
