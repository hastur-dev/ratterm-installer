# direnv

Environment switcher for the shell that loads/unloads environment variables based on current directory.

## Technology

- **Tool**: direnv
- **Category**: Shell, Environment
- **Language**: Go
- **Purpose**: Automatically load directory-specific environment variables

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to pacman (Arch Linux)
