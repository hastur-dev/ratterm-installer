# age

Simple, modern, and secure file encryption tool with small explicit keys and no configuration options.

## Technology

- **Tool**: age (filippo.io/age)
- **Category**: Security, Encryption
- **Purpose**: File encryption using public-key or passphrase-based encryption

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to pacman (Arch Linux)
4. Falls back to `go install` if Go is available
