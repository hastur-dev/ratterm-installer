# typos

Source code spell checker.

## Technology

- **Tool**: typos
- **Category**: Linting
- **Language**: Rust
- **Purpose**: Find and correct typos in source code

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to GitHub releases binary download

## Notes

Supports x86_64 and aarch64 architectures.
