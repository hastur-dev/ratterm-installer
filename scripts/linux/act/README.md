# act

Local GitHub Actions runner that allows you to run your GitHub Actions workflows locally using Docker containers.

## Technology

- **Tool**: act (nektos/act)
- **Category**: CI/CD, DevOps
- **Purpose**: Run GitHub Actions locally for testing and development

## Installation Method

1. Attempts installation via `pacman` (Arch Linux)
2. Falls back to official install script from nektos/act repository
