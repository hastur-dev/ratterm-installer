# Packer

Tool for creating identical machine images for multiple platforms.

## Technology

- **Tool**: HashiCorp Packer
- **Category**: DevOps
- **Language**: Go
- **Purpose**: Build automated machine images

## Installation Method

1. Attempts installation via HashiCorp APT repository (Debian/Ubuntu)
2. Falls back to Homebrew
