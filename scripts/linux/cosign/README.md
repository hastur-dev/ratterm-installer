# Cosign

Container signing, verification, and storage in an OCI registry.

## Technology

- **Tool**: Cosign (Sigstore)
- **Category**: Security, Container
- **Language**: Go
- **Purpose**: Sign and verify container images and other artifacts

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to direct binary download from GitHub releases
