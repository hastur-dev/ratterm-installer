# git-cliff

Highly customizable changelog generator following conventional commits.

## Technology

- **Tool**: git-cliff
- **Category**: Git, DevOps
- **Language**: Rust
- **Purpose**: Generate changelogs from git history

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to direct binary download from GitHub releases
