# Redis

In-memory data structure store.

## Technology

- **Tool**: Redis
- **Category**: Database
- **Language**: C
- **Purpose**: In-memory key-value store, cache, and message broker

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum with EPEL (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)

## Notes

- Service is automatically enabled and started via systemd
- Default port: 6379
- Includes both `redis-server` and `redis-cli`
