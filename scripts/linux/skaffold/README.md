# Skaffold

Local Kubernetes development tool.

## Technology

- **Tool**: Skaffold
- **Category**: Kubernetes
- **Language**: Go
- **Purpose**: Build and deploy applications to Kubernetes with a single command

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to direct binary download from Google Cloud Storage

## Notes

Supports amd64 and arm64 architectures.
