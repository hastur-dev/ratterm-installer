# ImageMagick

Software suite for creating, editing, and converting images.

## Technology

- **Tool**: ImageMagick
- **Category**: Image Processing
- **Language**: C
- **Purpose**: Image manipulation and conversion

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
