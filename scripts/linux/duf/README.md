# duf

Disk Usage/Free utility with a better output format than df.

## Technology

- **Tool**: duf
- **Category**: System Utilities
- **Language**: Go
- **Purpose**: User-friendly disk space information

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to snap
3. Falls back to dnf (Fedora/RHEL)
4. Falls back to pacman (Arch Linux)
5. Falls back to `go install`
