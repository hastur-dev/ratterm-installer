# wget

Network downloader utility.

## Technology

- **Tool**: GNU Wget
- **Category**: Network Utilities
- **Language**: C
- **Purpose**: Download files from the web via HTTP, HTTPS, and FTP

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
