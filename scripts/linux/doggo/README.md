# doggo

Command-line DNS client for humans with DNS-over-HTTPS and DNS-over-TLS support.

## Technology

- **Tool**: doggo
- **Category**: Networking, DNS
- **Language**: Go
- **Purpose**: Modern DNS client with DoH/DoT support

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to snap
3. Falls back to `go install`
