# Ansible

Agentless IT automation tool for configuration management, application deployment, and task automation.

## Technology

- **Tool**: Ansible
- **Category**: Automation, DevOps, Configuration Management
- **Language**: Python
- **Purpose**: Automate infrastructure provisioning and application deployment

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to pip3
4. Falls back to Homebrew
