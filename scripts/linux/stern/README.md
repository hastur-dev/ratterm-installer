# stern

Multi pod and container log tailing for Kubernetes.

## Technology

- **Tool**: stern
- **Category**: Kubernetes
- **Language**: Go
- **Purpose**: Tail multiple pods and containers in Kubernetes

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to GitHub releases binary download

## Notes

Supports amd64 and arm64 architectures.
