# mise

Polyglot runtime manager (formerly rtx).

## Technology

- **Tool**: mise
- **Category**: Version Manager
- **Language**: Rust
- **Purpose**: Manage multiple language runtimes (Node, Python, Ruby, etc.)

## Installation Method

Uses the official install script from mise.run.

## Notes

After installation, add to your shell config:
- Bash: `eval "$(mise activate bash)"`
- Zsh: `eval "$(mise activate zsh)"`
