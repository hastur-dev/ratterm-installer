# pipx

Install and run Python applications in isolated environments.

## Technology

- **Tool**: pipx
- **Category**: Package Manager
- **Language**: Python
- **Purpose**: Install Python CLI tools in isolated virtual environments

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to Homebrew
4. Falls back to pip3 user install

## Notes

Runs `pipx ensurepath` after installation to add to PATH.
