# htop

Interactive process viewer for Unix systems.

## Technology

- **Tool**: htop
- **Category**: System Monitoring
- **Language**: C
- **Purpose**: Interactive process viewer and system monitor

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS with EPEL)
4. Falls back to pacman (Arch Linux)
