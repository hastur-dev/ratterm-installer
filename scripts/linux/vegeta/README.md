# Vegeta

HTTP load testing tool and library.

## Technology

- **Tool**: Vegeta
- **Category**: Testing
- **Language**: Go
- **Purpose**: HTTP load testing with constant request rate

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to Go install
3. Falls back to GitHub releases binary download

## Notes

Supports amd64 and arm64 architectures.
