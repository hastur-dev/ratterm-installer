# Rust

Systems programming language focused on safety, speed, and concurrency.

## Technology

- **Tool**: Rust (rustc, cargo, rustup)
- **Category**: Programming Language
- **Purpose**: Build reliable and efficient software

## Installation Method

1. Uses official rustup installer from rustup.rs
2. Installs rustc, cargo, and rustup

## Configuration

After installation, run `source ~/.cargo/env` or restart your shell.
