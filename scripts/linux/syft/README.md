# Syft

CLI tool for generating Software Bill of Materials (SBOM).

## Technology

- **Tool**: Syft
- **Category**: Security
- **Language**: Go
- **Purpose**: Generate SBOMs from container images and filesystems

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to official install script from Anchore
