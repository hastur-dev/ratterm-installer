# dust

More intuitive version of du (disk usage) with a visual tree output.

## Technology

- **Tool**: dust (du-dust)
- **Category**: System Utilities
- **Language**: Rust
- **Purpose**: Visualize disk usage with an intuitive tree structure

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to pacman (Arch Linux)
4. Falls back to cargo install
