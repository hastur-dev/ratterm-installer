# Poetry

Python dependency management and packaging made easy.

## Technology

- **Tool**: Poetry
- **Category**: Package Manager
- **Language**: Python
- **Purpose**: Manage Python project dependencies and virtual environments

## Installation Method

Uses the official Poetry install script from install.python-poetry.org.

## Notes

Installs to `/usr/local` by default.
