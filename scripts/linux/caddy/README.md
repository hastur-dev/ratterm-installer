# Caddy

Fast, multi-platform web server with automatic HTTPS.

## Technology

- **Tool**: Caddy
- **Category**: Web Server
- **Language**: Go
- **Purpose**: Web server with automatic TLS certificate management

## Installation Method

1. Attempts installation via apt-get with official Caddy repository
2. Falls back to dnf with Caddy COPR repository
3. Falls back to Homebrew
