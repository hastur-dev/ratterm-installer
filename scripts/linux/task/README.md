# Task

Task runner / simpler Make alternative written in Go.

## Technology

- **Tool**: Task (go-task)
- **Category**: Build Tools
- **Language**: Go
- **Purpose**: Run tasks defined in a Taskfile.yml

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to Snap
3. Falls back to official install script
