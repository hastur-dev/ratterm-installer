# AWS CLI

Amazon Web Services Command Line Interface for managing AWS services from the terminal.

## Technology

- **Tool**: AWS CLI
- **Category**: Cloud, DevOps
- **Language**: Python
- **Purpose**: Manage AWS resources and services via command line

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
5. Falls back to official AWS CLI installer
