# dprint

Pluggable and configurable code formatter supporting multiple languages.

## Technology

- **Tool**: dprint
- **Category**: Formatting
- **Language**: Rust
- **Purpose**: Fast code formatting for TypeScript, JavaScript, JSON, Markdown, and more

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to cargo install
3. Falls back to official install script
