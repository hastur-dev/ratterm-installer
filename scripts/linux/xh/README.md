# xh

Friendly and fast tool for sending HTTP requests.

## Technology

- **Tool**: xh
- **Category**: HTTP Client
- **Language**: Rust
- **Purpose**: Send HTTP requests with HTTPie-compatible syntax

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to GitHub releases binary download

## Notes

Supports x86_64 and aarch64 architectures.
