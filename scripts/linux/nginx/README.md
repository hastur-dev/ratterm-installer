# nginx

High-performance HTTP and reverse proxy server.

## Technology

- **Tool**: nginx
- **Category**: Web Server
- **Language**: C
- **Purpose**: Serve web content, reverse proxy, and load balancer

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum with EPEL (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)

## Notes

- Service is automatically enabled and started via systemd
- Default port: 80
- Configuration: /etc/nginx/nginx.conf
