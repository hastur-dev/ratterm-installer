# Minikube

Local Kubernetes cluster for development and testing.

## Technology

- **Tool**: Minikube
- **Category**: Kubernetes
- **Language**: Go
- **Purpose**: Run a single-node Kubernetes cluster locally

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to direct binary download from Google Cloud Storage

## Notes

Supports amd64 and arm64 architectures.
