# fnm

Fast and simple Node.js version manager built in Rust.

## Technology

- **Tool**: fnm (Fast Node Manager)
- **Category**: Version Manager
- **Language**: Rust
- **Purpose**: Manage multiple Node.js versions

## Installation Method

1. Attempts installation via pacman (Arch Linux)
2. Falls back to cargo install
3. Falls back to official install script
