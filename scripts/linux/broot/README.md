# broot

Interactive directory tree navigator and file manager with fuzzy search.

## Technology

- **Tool**: broot
- **Category**: File Manager, Navigation
- **Language**: Rust
- **Purpose**: Navigate and search directory trees interactively

## Installation Method

1. Attempts installation via pacman (Arch Linux)
2. Falls back to cargo install
