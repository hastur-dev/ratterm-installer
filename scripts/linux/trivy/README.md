# Trivy

Comprehensive security scanner.

## Technology

- **Tool**: Trivy
- **Category**: Security
- **Language**: Go
- **Purpose**: Scan container images, file systems, and git repos for vulnerabilities

## Installation Method

1. Attempts installation via apt-get with Aqua Security repository (Debian/Ubuntu)
2. Falls back to Homebrew
3. Falls back to Snap
