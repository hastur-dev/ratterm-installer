# mdBook

Create books from Markdown files.

## Technology

- **Tool**: mdBook
- **Category**: Documentation
- **Language**: Rust
- **Purpose**: Build documentation websites from Markdown

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to GitHub releases binary download

## Notes

Supports x86_64 and aarch64 architectures.
