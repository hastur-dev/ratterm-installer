# difftastic

Structural diff tool that understands syntax, not just text.

## Technology

- **Tool**: difftastic (difft)
- **Category**: Diff, Git
- **Language**: Rust
- **Purpose**: Language-aware structural diffs for code review

## Installation Method

1. Attempts installation via cargo
2. Falls back to pacman (Arch Linux)
3. Falls back to direct binary download from GitHub releases
