# shfmt

Shell script formatter.

## Technology

- **Tool**: shfmt
- **Category**: Formatting
- **Language**: Go
- **Purpose**: Format shell scripts (bash, sh, mksh)

## Installation Method

1. Attempts installation via Snap
2. Falls back to Homebrew
3. Falls back to Go install
4. Falls back to GitHub releases binary download

## Notes

Supports amd64 and arm64 architectures.
