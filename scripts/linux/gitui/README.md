# gitui

Blazingly fast terminal UI for git written in Rust.

## Technology

- **Tool**: gitui
- **Category**: Git, TUI
- **Language**: Rust
- **Purpose**: Terminal-based Git client with keyboard-driven interface

## Installation Method

1. Attempts installation via pacman (Arch Linux)
2. Falls back to cargo install
