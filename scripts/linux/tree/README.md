# tree

Display directory structure as a tree.

## Technology

- **Tool**: tree
- **Category**: CLI Utilities
- **Language**: C
- **Purpose**: Recursively list directories in a tree format

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
