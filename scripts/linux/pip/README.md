# pip

Package installer for Python.

## Technology

- **Tool**: pip
- **Category**: Package Manager
- **Language**: Python
- **Purpose**: Install and manage Python packages

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)

## Notes

Installs `python3-pip` (or `python-pip` on Arch).
