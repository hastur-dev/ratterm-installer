# gping

Ping with a graph visualization in the terminal.

## Technology

- **Tool**: gping
- **Category**: Networking
- **Language**: Rust
- **Purpose**: Visualize ping latency over time

## Installation Method

1. Attempts installation via cargo
2. Falls back to pacman (Arch Linux)
3. Falls back to apt-get with azlux repository
4. Falls back to direct binary download from GitHub releases
