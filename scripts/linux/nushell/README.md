# Nushell

Modern shell with structured data pipelines.

## Technology

- **Tool**: Nushell (nu)
- **Category**: Shell
- **Language**: Rust
- **Purpose**: Shell that treats data as structured tables

## Installation Method

1. Attempts installation via Cargo
2. Falls back to pacman (Arch Linux)
3. Falls back to GitHub releases binary download
