# yq

Lightweight and portable command-line YAML processor.

## Technology

- **Tool**: yq
- **Category**: Data Processing
- **Language**: Go
- **Purpose**: Query and manipulate YAML, JSON, and XML files

## Installation Method

1. Attempts installation via Snap
2. Falls back to apt-get with GitHub releases binary (Debian/Ubuntu)
3. Falls back to dnf or GitHub releases binary (Fedora/RHEL)
4. Falls back to pacman (Arch Linux)
