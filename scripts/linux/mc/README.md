# mc (MinIO Client)

Command-line tool for managing cloud storage services.

## Technology

- **Tool**: MinIO Client (mc)
- **Category**: Cloud Storage
- **Language**: Go
- **Purpose**: Manage S3-compatible object storage from the command line

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to direct binary download from MinIO

## Notes

Supports amd64 and arm64 architectures.
