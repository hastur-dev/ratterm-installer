# GoReleaser

Release automation tool for Go projects.

## Technology

- **Tool**: GoReleaser
- **Category**: DevOps, CI/CD
- **Language**: Go
- **Purpose**: Automate building and publishing Go binaries

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to snap
3. Falls back to `go install`
