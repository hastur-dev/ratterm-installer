# Atuin

Magical shell history that syncs, searches, and backs up your command history across machines.

## Technology

- **Tool**: Atuin
- **Category**: Shell, Productivity
- **Language**: Rust
- **Purpose**: Enhanced shell history with sync, search, and statistics

## Installation Method

1. Attempts installation via pacman (Arch Linux)
2. Falls back to cargo install
