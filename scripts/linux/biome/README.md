# Biome

Fast formatter and linter for JavaScript, TypeScript, JSX, and JSON, designed as a Prettier and ESLint replacement.

## Technology

- **Tool**: Biome (formerly Rome)
- **Category**: Linting, Formatting
- **Language**: Rust
- **Purpose**: JavaScript/TypeScript formatting and linting

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to npm global install
3. Falls back to cargo install
