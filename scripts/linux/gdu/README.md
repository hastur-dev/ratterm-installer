# gdu

Fast disk usage analyzer with console interface.

## Technology

- **Tool**: gdu (Go Disk Usage)
- **Category**: System Utilities
- **Language**: Go
- **Purpose**: Analyze disk usage with an interactive ncurses interface

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to pacman (Arch Linux)
4. Falls back to `go install`
