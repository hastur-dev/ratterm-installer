# Pandoc

Universal document converter.

## Technology

- **Tool**: Pandoc
- **Category**: Document Conversion
- **Language**: Haskell
- **Purpose**: Convert between markup formats (Markdown, HTML, LaTeX, etc.)

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
