# FFmpeg

Complete, cross-platform solution for recording, converting, and streaming audio and video.

## Technology

- **Tool**: FFmpeg
- **Category**: Multimedia
- **Language**: C
- **Purpose**: Audio/video encoding, decoding, transcoding, and streaming

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
