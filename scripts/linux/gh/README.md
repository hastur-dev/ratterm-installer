# GitHub CLI

GitHub's official command-line tool for managing repositories, issues, PRs, and more.

## Technology

- **Tool**: gh (GitHub CLI)
- **Category**: DevOps, Git
- **Language**: Go
- **Purpose**: Interact with GitHub from the command line

## Installation Method

1. Attempts installation via apt-get with GitHub's official repository
2. Falls back to dnf/yum with GitHub's RPM repository
3. Falls back to pacman (Arch Linux)
