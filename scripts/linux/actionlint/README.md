# actionlint

Static checker for GitHub Actions workflow files that catches common errors and best practice violations.

## Technology

- **Tool**: actionlint (rhysd/actionlint)
- **Category**: Linting, CI/CD
- **Purpose**: Validate GitHub Actions workflow YAML files

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to direct binary download from GitHub releases (amd64/arm64)
