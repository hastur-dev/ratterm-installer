# SOPS

Simple and flexible tool for managing secrets.

## Technology

- **Tool**: SOPS (Secrets OPerationS)
- **Category**: Security
- **Language**: Go
- **Purpose**: Encrypt and decrypt files using AWS KMS, GCP KMS, Azure Key Vault, age, or PGP

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to GitHub releases binary download

## Notes

Supports amd64 and arm64 architectures.
