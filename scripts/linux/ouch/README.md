# ouch

Painless compression and decompression tool.

## Technology

- **Tool**: ouch
- **Category**: Archive Utilities
- **Language**: Rust
- **Purpose**: Compress and decompress files with automatic format detection

## Installation Method

1. Attempts installation via pacman (Arch Linux)
2. Falls back to Cargo installation
