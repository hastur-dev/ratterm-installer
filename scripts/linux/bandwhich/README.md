# bandwhich

Terminal bandwidth utilization tool that displays current network utilization by process, connection, and remote IP/hostname.

## Technology

- **Tool**: bandwhich
- **Category**: Networking, Monitoring
- **Language**: Rust
- **Purpose**: Real-time bandwidth monitoring per process and connection

## Installation Method

1. Attempts installation via cargo
2. Falls back to pacman (Arch Linux)
3. Falls back to direct binary download from GitHub releases
