# hadolint

Dockerfile linter that validates inline bash and helps build best practice Docker images.

## Technology

- **Tool**: hadolint
- **Category**: Linting, Docker
- **Language**: Haskell
- **Purpose**: Lint Dockerfiles for best practices

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to direct binary download from GitHub releases
