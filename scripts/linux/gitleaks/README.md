# gitleaks

Secret scanning tool for git repositories to detect hardcoded secrets.

## Technology

- **Tool**: gitleaks
- **Category**: Security
- **Language**: Go
- **Purpose**: Detect secrets, passwords, and API keys in git repos

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to direct binary download from GitHub releases
