# lazygit

Terminal UI for git commands.

## Technology

- **Tool**: lazygit
- **Category**: Git, TUI
- **Language**: Go
- **Purpose**: Simple terminal UI for git with keyboard shortcuts

## Installation Method

1. Attempts installation via apt-get with PPA (Ubuntu)
2. Falls back to dnf with COPR (Fedora)
3. Falls back to yum with COPR (RHEL)
4. Falls back to pacman (Arch Linux)
