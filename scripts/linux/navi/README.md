# navi

Interactive cheatsheet tool for the command-line.

## Technology

- **Tool**: navi
- **Category**: CLI Utilities
- **Language**: Rust
- **Purpose**: Browse and execute command snippets from cheatsheets

## Installation Method

1. Attempts installation via Homebrew
2. Falls back to Cargo
3. Falls back to GitHub releases binary download

## Notes

Supports x86_64 and aarch64 architectures.
