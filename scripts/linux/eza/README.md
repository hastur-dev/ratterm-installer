# eza

Modern replacement for ls with Git integration and rich formatting.

## Technology

- **Tool**: eza (fork of exa)
- **Category**: CLI Utilities
- **Language**: Rust
- **Purpose**: Enhanced file listing with colors, icons, and Git status

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
