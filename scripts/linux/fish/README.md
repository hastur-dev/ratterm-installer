# fish

Friendly interactive shell with syntax highlighting, autosuggestions, and web-based configuration.

## Technology

- **Tool**: fish shell
- **Category**: Shell
- **Language**: C++
- **Purpose**: User-friendly command line shell

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
