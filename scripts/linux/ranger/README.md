# ranger

Console file manager with VI key bindings.

## Technology

- **Tool**: ranger
- **Category**: File Manager
- **Language**: Python
- **Purpose**: Terminal-based file manager with preview support

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to yum (older RHEL/CentOS)
4. Falls back to pacman (Arch Linux)
