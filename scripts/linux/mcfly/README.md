# mcfly

Intelligent shell history search with neural network.

## Technology

- **Tool**: mcfly
- **Category**: Shell Enhancement
- **Language**: Rust
- **Purpose**: Smart shell history search using contextual awareness

## Installation Method

1. Attempts installation via Cargo
2. Falls back to pacman (Arch Linux)
3. Falls back to GitHub releases binary download

## Notes

After installation, add to your shell config:
- Bash: `eval "$(mcfly init bash)"`
- Zsh: `eval "$(mcfly init zsh)"`
