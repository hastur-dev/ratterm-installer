# lsd

LSDeluxe - Modern replacement for ls with colors and icons.

## Technology

- **Tool**: lsd (LSDeluxe)
- **Category**: CLI Utilities
- **Language**: Rust
- **Purpose**: Enhanced file listing with icons and colors

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to pacman (Arch Linux)
4. Falls back to cargo install
