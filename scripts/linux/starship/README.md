# Starship

Cross-shell prompt that is minimal, blazing fast, and customizable.

## Technology

- **Tool**: Starship
- **Category**: Shell, Prompt
- **Language**: Rust
- **Purpose**: Beautiful and informative shell prompt

## Installation Method

1. Uses official install script from starship.rs

## Configuration

Add to your shell config:
- Bash: `eval "$(starship init bash)"`
- Zsh: `eval "$(starship init zsh)"`
- Fish: `starship init fish | source`
