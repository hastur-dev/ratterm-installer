# delta

Syntax-highlighting pager for git, diff, and grep output.

## Technology

- **Tool**: delta (git-delta)
- **Category**: Git, CLI Utilities
- **Language**: Rust
- **Purpose**: Beautiful diffs with syntax highlighting and line numbers

## Installation Method

1. Attempts installation via apt-get (Debian/Ubuntu)
2. Falls back to dnf (Fedora/RHEL)
3. Falls back to pacman (Arch Linux)
4. Falls back to cargo install
