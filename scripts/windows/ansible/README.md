# Ansible

IT automation platform.

## Technology

- **Tool**: Ansible
- **Category**: DevOps
- **Language**: Python
- **Purpose**: Configuration management and application deployment

## Installation Method

Installs via pip (requires WSL or Python).

## Notes

Limited native Windows support; typically run via WSL.
